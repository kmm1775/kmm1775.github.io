Štruktúra

Stránka pozostáva celkovo zo 3 layoutov, pričom jeden z týchto layout-ov je hlavný layout od ktorého "dedia" zvyšné 2 layouti - stránka bez bočného menu a stránka s bočným menu.

Po obsahovej stránke je projekt rozdelení do statických HTML stránok kt. tvoria podstránky projektu. Sú to:

index.html - hlavná stránka kt. obsahuje novinky. Každá novinka je reprezentovaná markdown súborom, kt. obsahuje popis novinky. Novinky sa pridávajú prostredníctvom cyklu.
omne.html - stránka obsahujúca môj osobný profil   tj. developerský profil a koníčky. Jednotlivé položky - programovacie jazyky v developerskom profile sú generované prostredníctvom dátových sád obsiahnutých v súbore progjazyky.yml
projekty.html - hlavná stránka projektov.
projekt1.html - stránka pre projekt1 obsahujúca dokumentáciu a popis projektu. Po pridaní projektov 2 a 3 pribudnú unikátne stránky aj pre tieto projekty.
ostranke.html - obsahuje základné informácie o stránke.

Použité prvky

Tagy a filtre

{% raw %}{% comment %}{% endraw %} na odkomentovanie určitých častí kódu ako aj pre pridanie komentárov opisujúcich funkčnosť, použité napr. v súbore index.html pri generovaní noviniek.

{% raw %}{% for ... %}{% endraw %} na generovanie noviniek resp. ostatných dátových sád repetetívne. Použité napríklad pri výpise používaných programovacích jazykov v súbore omne.html.

{% raw %}{% capture %}{% endraw %} pre posúvanie vkladaných súborov prostredníctvom {% raw %}{% include %}{% endraw %} do markdown filtra {% raw %}{% ... | markdownify %}{% endraw %}. Použité napríklad pri vkladaní noviniek v súbore index.html, kde je novinka vo formáte markdown najskôr vložená do šablóny a následne je jej obsah presunutý do premennej a vygenerovaný prostredníctvom filtra {% raw %}{% ... | markdownify %}{% endraw %}

filter Markdownify slúži na spracovanie markdown formátovania.
filter Date to Long String pre vypísanie aktuálneho dátumu.

Premenné

Sú použité globálne premenné page.title a site.html_pages pre vypísanie zoznamu všetkých HTML stránok s ktorých je projekt tvorený. Ďalej sú použité custom premenné za účelom vykreslovania dátových sád a zisťovania prítomnosti projektov a vykreslovania ich liniek.

Plugin

Ako plugin som použil plugin Emoji for Jekyll, ktorý umožnuje interpretovať emoji ikony v textoch.

Pri tvorbe stránky som použil existujúce štýly pre šablónu z TEMPLATEDCO, ktorú som mierne upravil. Obrázky (logá) programovacích jazykov som prevzal z internetu.
