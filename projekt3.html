Popis riešenia

Úvod

Zadanie obsahuje ukážkový XML dokument transformovaný pomocou XSLT do jednotlivých HTML slajdov. Použitá schéma bola RELAXNG.

Použité elementy

Znaková postupnosť v zátvorkách vyjadruje početnosť: >n - n a viac, m/n - práve m alebo n, n - práve n

  • element <presentation> - obaluje celú prezentácia

METADÁTA

  • element <meta> - obaľuje sekciu metadát (1)
  • element <authors> - obsahuje jednotlivých autorov prezentácie (1)
  • element <author> - zaobaľuje informácie o jednotlivcovi - autorovi (>1)
  • element <name> - obsahuje meno autora (1)
  • element <contact> - obsahuje kontakt na autora - jeho e-mailovú adresu (0/1)
  • element <creation> - obsahuje dodatočné metadáta o prezentácií (1)
  • element <created> - obsahuje dátum a čas vytvorenia (1)
  • element <lastedit> - obsahuje dátum a čas poslednej zmeny dokumentu (1)
  • element <editor> - obsahuje dáta o použitom editore napr. powerpoint (1)

OBSAH / ŠTRUKTÚRA

  • element <slide> - element definujúci slajd prezentácie, môže obsahovať atribúty: (1)
    • id: vyjadrujúci typ slajdu (titulný, obyčajný, finálny)
    • transition: typ prechodového efektu (niekoľko ukážkých typov ako fade, wipe alebo none)
    • width: šírku plátna
    • height: výšku plátna
    • font-size: veľkosť fontov v slajde
  • element <title> - element vyjadruje titulok/nadpis slajdu, je možné v ňom definovať inline elementy (1)
  • element <subtitle> - definuje podtitulok slajdu, je možné v ňom definovať inline elementy (0/1)
  • element <background> - definuje obrázok pozadia slajdu. Má atribúty: (1)
    • src : zdrojový obrázok
    • repeat : štýl opakovania vzoru pozadia, podobne ako v CSS
  • element <content> - definuje obsahovú stránku slajdu, v ňom sú vnorené elementy: (1)
  • <textbox> - textový odstavec (>0)
  • <header> - nadpis (>0)
  • <list> a <item> - zoznam a položka zoznamu (>0)
  • <picture> - obrázok (párny element umožnujúci vložiť aj jeho popis) (>0)

Elementy v rámci elementu content je možné absolútne pozicovať lubovoľný element tak ako v skutočnej prezentácií. Toto je umožnené pomocou atribútov posx a posy. Elementy v rámci elementu obsah môžu obsahovať inline elementy.

INLINE ELEMENTY

  • <bold> - hrubé zvýraznenie
  • <emphasis> - sémantické zvýrazenie
  • <italic> - kurzíva
  • <color> - s parametrom which definujúcim meno farby alebo jej čiselenú hodnotu.
  • <comment> - komentár

Početnosť: žiadny z elementov nie je povinný.

VALIDÁCIA

Ako schéma dokumentu bola používa nesimplifikovaná schéma typu RELAXNG. Ako validátor bol použitý jing.
